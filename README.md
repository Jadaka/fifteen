# Fifteen

A fifteen puzzle game built for the browser.

# Development

```bash
# create the database by running scripts/create-database.sql
mysql -u [user] -p[password] < scripts/schema.sql

# install dependencies
yarn

# start dev server
yarn start

# run tests
yarn test

# create production build
yarn build
```
